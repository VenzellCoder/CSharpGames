# Tower Defence
"Tower Defence" с редактором волн.

## Редактор волн
Редактируемые параметры:
-количество волн на уровне
-количество этапов в волне
-тип юнитов в волне
-интервал между юнитами на этапе
-задержка между этапами в волне
